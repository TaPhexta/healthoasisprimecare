# Health Oasis - IV Drip Therapy & Wellness Clinic Website

## Overview
Health Oasis is a modern, responsive website for a premier IV drip therapy and wellness clinic. The website showcases the clinic's services, team, and provides an easy way for clients to get in touch.

## Pages
- **Home (index.html)**: Welcome page with hero carousel, core services overview, and client testimonials
- **Services (services.html)**: Complete listing of 24 IV therapy treatments with filtering by category and price range
- **About (about.html)**: Clinic mission, team members, and client reviews
- **Contact (contact.html)**: Contact form with validation and clinic information

## Design Theme
- **Primary Color**: Olive Green (#556B2F)
- **Secondary Color**: Black (#000000)
- **Accent Color**: Beige (#F5F5DC)
- **Gold Accent**: Gold (#DAA520)

## Key Features
- Responsive design for all device sizes
- Auto-rotating hero carousel (3-second intervals)
- Reveal-on-scroll animations
- Service filtering by category and price range
- Dropdown navigation menu on hover
- Form validation on contact page
- Mobile-friendly hamburger menu

## Technical Stack
- React 18 (CDN)
- TailwindCSS
- Lucide Icons
- Vanilla JavaScript for interactions
